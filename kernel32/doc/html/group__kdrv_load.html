Load and initialize drivers for given device ID.

Parameters
*deviceIdDevice ID string
**driversDynamically allocated table of loaded driver objects pointers
*driverCountNumber of loaded drivers
Returns
Status code. **drivers is NULL when function fails.
Warning
This function fails when at least one required driver was not loaded.
Note
Freeing the **drivers table must be done by the caller.
This function reuses already loaded drivers.
